Detailed Description

A trasfomation from a single numeric value to another numeric value or color.

As input value it accepts only values which fits into the input range of values. If input range values are same, then only a single value is accepted into this trasforamtion. Output depens on type of output; numeric value or color. Numeric output type can be determined by output range or by mahtematical expression.
