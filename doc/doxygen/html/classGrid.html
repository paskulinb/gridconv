Detailed Description

Class representing a single GIS raster file.

It supports opening 'ESRI ASCII grid' and 'ESRI float grid' file formats.

int Grid::openGrid ( std::string  filepath)

Open a raster data file.

Two GIS raster fileformats are supported: ESRI ASCII grid and ESRI float grid. The fileformat can be specified by file extension in 'filepath' parameter. ASCII grid has .txt extension, Float grid has .flt extensiton. If no extension is defined, the ASCII grid presence will be checked first. If not found, then Float file will be opened if existent.

Parameters
filepathpath to file
Returns
1: OK
0: ERROR
